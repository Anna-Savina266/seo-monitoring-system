Проект: автоматический мониторинг SEO-показателей сайта.

Проблема
SEO-аудит сейчас делается вручную (PageSpeed, GSC, Яндекс.Вебмастер) — 2–3 дня на сбор, данные быстро устаревают, деградация обнаруживается с опозданием.

Решение
Система собирает метрики по расписанию через API, хранит историю, сравнивает с порогами и алертит при деградации.

Артефакты
- BPMN AS-IS/TO-BE
- Контекстная диаграмма и DFD
- Макет дашборда
- Требования (функциональные/ нефункциональные), пороги алертов, глоссарий
- Полный отчёт

Стейкхолдеры
SEO-специалист · маркетинг-директор · разработчик · контент-менеджер · IT-команда

Интеграции
Google PageSpeed Insights API · Google Search Console API · Яндекс.Вебмастер API · Jira/YouTrack · Email/Telegram
